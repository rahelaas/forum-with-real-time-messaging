# **_Real-Time Forum Project_**

## **How to run:**
```
You need to run both back-end and front-end servers
```
### To run back-end server (_GO_):
```
cd server
then "go run ."
```

### To run front-end server (_VUEJS_):
```
cd front-end
then "npm install"
then "npm run serve"
```
## When both servers are up & running visit
```
http://localhost:8080
```
